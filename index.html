SnowFall

by: Jason Tablas

This game was made for my level 2 project at The League of Amazing Programmers.
Click here to download and play!

1. Why I Made This Game

I decided to make this game because it was one that I would see myself playing. One that was simple enough to entertain me for a couple of minutes but also challenging enough to make me want to get a high score every time I played.

2. What I Learned

I learned that you can always improve what you thought was perfect. They helped clean up some bugs and refine the movements.

3. Architecture of My Game

There are two main components, the trees and the skier. The skier has to avoid the trees as they are coming towards him, increasing in numbers making the game more difficult. The trees are chaos while the skier is moving with grace.

4. Technical Implementation

I am proud of being able to decrease the spawn time of the trees as time went on in the game. I wanted to make this game based off on time and I learned some new methods that were able to help me.I had to create time variables or "longs", along with "double" and "float". I made it so that every 5 seconds that go by in the game, the enemy spawn time decreases.

5. Most Challenging Part

My game lost some code right after I finished it so I had to refix some of the past problems that were tricky, such as the time set up. I wanted to restart the time after every death but it stayed at zero. It took some time to fix it.

6. Future Fixes and Updates

I would like to add a way to chop through the trees in case you are in a tight spot. I feel like adding more power-ups which would grant advantages for a limited time.
